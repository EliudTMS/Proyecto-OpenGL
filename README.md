Creeper 3D en OpenGL

Este proyecto modela un Creeper de Minecraft en 3D utilizando OpenGL.
El personaje cuenta con dos animaciones continuas:

Caminar: el Creeper avanza con un movimiento básico de piernas y cuerpo.

Explosión simulada: después de un tiempo, el Creeper comienza a hincharse, imitando la animación previa a la explosión típica del juego.

Nota: El Creeper no explota, simplemente la animación se reinicia para comenzar de nuevo el ciclo.

Este proyecto sirve como ejemplo de modelado, animación y control de estados en OpenGL.
